# Team member:

## Usage

Locate folder of repository, run the following command in terminal:

``` bash
pod install 
```
After this open `Bin Finder.xcworkspace`


## Requirements

- Xcode 10 or higher
- Minimum iOS deployment target of 10.0 or higher
- CocoaPods 1.5 or higher
